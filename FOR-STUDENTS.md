# 🎓 Guide pour les Étudiants

Salut ! Ce repo contient tous les skills dont tu as besoin pour coder comme un pro avec Claude.

## 🚀 Démarrage Ultra-Rapide (5 minutes)

### Étape 1 : Clone le Repo

```bash
git clone https://github.com/butlucratif/claude-skills-collection.git
cd claude-skills-collection
```

### Étape 2 : Installe les Skills

```bash
# Copie TOUS les skills
cp -r * ~/.claude/skills/

# Ou copie juste ceux qui t'intéressent
cp -r test-driven-development ~/.claude/skills/
cp -r brainstorming ~/.claude/skills/
```

### Étape 3 : Test !

Ouvre Claude Code et essaye :

```
Je veux créer une fonction qui valide un email.
Utilise test-driven-development pour m'aider.
```

## 🎯 Exercices Pratiques

### Exercice 1 : Premier Skill
**Objectif :** Comprendre comment fonctionne un skill

1. Lis le fichier `test-driven-development/SKILL.md`
2. Essaye de l'utiliser sur une fonction simple
3. Note ce qui a marché et ce qui n'a pas marché

### Exercice 2 : Combiner des Skills
**Objectif :** Apprendre à combiner plusieurs skills

1. Choisis un petit projet
2. Utilise `brainstorming` pour planifier
3. Utilise `test-driven-development` pour implémenter
4. Utilise `verification-before-completion` pour vérifier

### Exercice 3 : Créer Ton Skill
**Objectif :** Personnaliser ton workflow

1. Identifie une tâche que tu répètes souvent
2. Utilise `writing-skills` pour créer un skill
3. Teste-le sur plusieurs projets
4. Améliore-le progressivement

## 💪 Tips pour Réussir

### 1. Commence Petit
Ne te lance pas directement dans tous les skills. Commence par 2-3 et maîtrise-les.

### 2. Pratique Régulièrement
Utilise les skills sur tous tes projets, même les petits.

### 3. Expérimente
N'hésite pas à adapter les skills à ta façon de travailler.

### 4. Partage
Aide tes camarades, partage tes découvertes, contribue au repo.

### 5. Documente
Note ce qui marche bien pour toi, crée tes propres guides.
